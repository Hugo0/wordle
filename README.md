# Wordle Global

The world's word game — 80+ languages, 1.6 million users across 200 countries.

**Play now:** [wordle.global](https://wordle.global)

## About

Wordle Global started in 2022 with a simple open-source mission: bring Wordle to every language in the world, not just English. What began as a weekend project grew into a daily word game played by 1.6 million people in 80+ languages — from Finnish to Te Reo Māori, Arabic to Esperanto.

Active development has moved to a private repository. This repo is maintained as a community hub for bug reports and feature requests, and preserves the original open-source codebase and contributor history.
