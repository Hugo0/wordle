# Polish Language Data - Sources and Licenses

This directory contains Polish language data compiled from multiple sources.

## Sources

### 1. SJP.pl - Słownik Języka Polskiego
- **URL**: https://sjp.pl/sl/odmiany/
- **License**: Apache 2.0 (selected from available options: GPL 2, LGPL 2.1, CC BY 4.0, Apache 2.0)
- **Usage**: Word forms and conjugations used to build the word list

### 2. FrequencyWords
- **URL**: https://github.com/hermitdave/FrequencyWords
- **License**: MIT
- **Usage**: Frequency data used to select common words

## Modifications

The word lists in this directory (`pl_5words.txt`, `pl_5words_supplement.txt`) are derived works:
- Combined data from both sources above
- Filtered to only 5-letter words
- Edited for quality and appropriateness
- Sorted by frequency and game suitability
